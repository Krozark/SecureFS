## [1.0.0] - 2024-12-04

### Added
- Initial release of SecureFS
- Transparent file encryption with AES-256-GCM
- Two-level encryption architecture
- SQLite metadata storage
- Integrity verification
- Thread-safe operations
- Optional caching
- Development mode
- Migration support
- Modern tooling with Ruff
